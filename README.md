# Price Comparise Tool (价格对比工具)

简短描述
---
Price Comparise Tool 是一个轻量级的静态网页前端，用于展示和比较商品价格（仓促命名为 "comparise"）。该仓库当前由纯 HTML 文件组成，适合作为价格对比界面原型或演示页。你可以直接打开页面查看界面，或把它部署到 GitHub Pages 等静态站点托管服务。
